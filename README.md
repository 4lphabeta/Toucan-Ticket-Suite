# Toucan-Ticket-Suite
An ahk script that automates/speeds up ticket entry and callbacks.

Run the installer once to setup the file structure automatically. The installer will make it so the program runs on PC startup and can be found in the Windows task tray.

Right clicking the tray icon when the program is running will give the following options: https://i.imgur.com/zGek6KO.png.
The top option will show you a list of pre-programmed hotkeys https://i.imgur.com/fJ1Gw3h.png.
The rest should be self explanatory and can be changed as needed.
